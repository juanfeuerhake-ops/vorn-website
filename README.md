# VORN — Ropa Atemporal

Sitio web de marca de ropa escandinava de lujo minimalista. Vanilla HTML, CSS y JavaScript — sin dependencias, sin frameworks.

## Estructura

```
vorn-website/
├── index.html   # Estructura y contenido
├── style.css    # Estilos (paleta sand/ink, tipografía editorial)
├── main.js      # Scroll reveal + interacciones
└── README.md
```

## Publicar en GitHub Pages

1. Sube el repositorio a GitHub
2. Ve a **Settings → Pages**
3. En *Source*, selecciona la rama `main` y la carpeta `/ (root)`
4. Guarda — GitHub Pages publicará el sitio en `https://<tu-usuario>.github.io/<nombre-repo>/`

## Tipografía

- **Cormorant Garamond** — titulares editoriales
- **Outfit** — texto funcional y navegación
